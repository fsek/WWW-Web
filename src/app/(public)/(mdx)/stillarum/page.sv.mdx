# Stillarum/Bönerum

Vid Lunds universitet är vi medvetna om att studenter har olika behov och kräver utrymmen som tillgodoser deras individuella välbefinnande och akademiska framgång. För att möta mångfalden i studentgruppen erbjuder universitetet Stillarum - trygga och lugna platser där studenterna kan samla sina tankar, meditera, ägna sig åt personlig bön eller helt enkelt ta en paus från det krävande universitetslivet.

Dessa utrymmen är också ovärderliga för neurodivergenta studenter som ibland kan tycka att universitetsmiljön är överväldigande och behöver ett lugnt utrymme för att omgruppera och ladda om.

Stillarum fungerar som en trygg plats för personer med olika bakgrund och trosuppfattning, vilket främjar inkludering och välbefinnande inom vår akademiska gemenskap.
